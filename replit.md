# نظام إدارة مكتب المحاماة - Law Firm Management System

## نظرة عامة
نظام إدارة متكامل لمكتب المحاماة مع واجهة عربية تدعم RTL. يتميز بتصميم رسمي فاخر بألوان الكحلي الداكن والذهبي.

## التقنيات المستخدمة
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Express.js, Node.js
- **UI Components**: Shadcn/UI
- **State Management**: React Context with localStorage persistence
- **Routing**: Wouter

## الأدوار والصلاحيات
### المسؤول/المحامي (Admin/Lawyer)
- عرض جميع القضايا
- تعديل الملفات
- إرسال للمراجعة
- اعتماد أو إعادة الملفات

### السكرتير (Secretary)
- إضافة طلبات جديدة
- توزيع الملفات على المحامين
- إغلاق وأرشفة الملفات الجاهزة

## دورة العمل (Workflow)
1. **جديد**: السكرتير يضيف طلب جديد
2. **قيد التنفيذ**: السكرتير يوزع على محامي
3. **مراجعة**: المحامي يرسل للمراجعة
4. **جاهز للتسليم**: المسؤول يعتمد الملف
5. **مغلق**: السكرتير يغلق ويؤرشف

## هيكل المشروع
```
client/
  src/
    pages/
      login.tsx       - صفحة تسجيل الدخول
      dashboard.tsx   - لوحة التحكم
      cases.tsx       - إدارة القضايا
    components/
      app-sidebar.tsx - الشريط الجانبي
    lib/
      auth-context.tsx   - إدارة المصادقة
      cases-context.tsx  - إدارة القضايا
      theme-provider.tsx - الوضع الداكن/الفاتح
server/
  routes.ts   - API endpoints
  storage.ts  - تخزين البيانات
shared/
  schema.ts   - تعريفات البيانات
```
