=== 1. Schema في shared/schema.ts ===
- enum NotificationType: task_reminder, case_delay, consultation_delay, general_alert, deadline_warning, assignment, escalation, response_request
- enum NotificationPriority: low, medium, high, urgent
- enum NotificationStatus: pending, sent, read, responded, escalated, archived
- enum ResponseType: completed, in_progress, need_more_time, noted

- interface Notification:
  id, type, priority, status
  title, message
  senderId, senderName
  recipientId (أو recipientIds[] للإرسال الجماعي)
  relatedType: "case" | "consultation" | "task" | null
  relatedId: معرف القضية/الاستشارة/المهمة
  isRead, readAt
  response: { type: ResponseType, message: string, respondedAt: Date } | null
  scheduledAt: Date | null (للجدولة)
  escalationLevel: number (0 = عادي، 1 = مصعّد مرة، 2 = مصعّد مرتين)
  escalatedTo: userId | null
  autoEscalateAfterHours: number (افتراضي 24)
  createdAt, updatedAt

- interface NotificationTemplate:
  id, name, title, message, type, priority

- interface UserNotificationPreferences:
  userId
  enableSound: boolean
  enableDesktop: boolean
  digestMode: "instant" | "daily" | "weekly"
  mutedTypes: NotificationType[]
  quietHoursStart: string | null (مثل "22:00")
  quietHoursEnd: string | null (مثل "08:00")

- أضف صلاحية canSendNotifications في permissions للأدوار: branch_manager, department_head, legal_supervisor

=== 2. أنشئ client/src/lib/notifications-context.tsx ===
قائمة الإشعارات مع localStorage والدوال:

إرسال:
- sendNotification(notification): إرسال إشعار فوري
- sendBulkNotification(recipientIds[], notification): إرسال جماعي
- scheduleNotification(notification, scheduledAt): جدولة إشعار
- sendToTeam(departmentId, notification): إرسال لقسم كامل

قراءة وإدارة:
- markAsRead(id): تحديد كمقروء
- markAllAsRead(): تحديد الكل كمقروء
- deleteNotification(id): حذف
- archiveOldNotifications(daysOld): أرشفة القديمة
- getUnreadCount(): عدد غير المقروءة
- getMyNotifications(filters): إشعاراتي مع فلترة

الردود:
- respondToNotification(id, responseType, message): رد على إشعار
- getNotificationResponses(senderId): ردود على إشعاراتي المرسلة

التصعيد:
- checkAndEscalate(): فحص الإشعارات غير المقروءة وتصعيدها
- escalateNotification(id, escalateToUserId): تصعيد يدوي
- getEscalatedNotifications(): الإشعارات المصعّدة

التفضيلات:
- getUserPreferences(userId): جلب تفضيلات المستخدم
- updateUserPreferences(userId, preferences): تحديث التفضيلات

القوالب:
- getTemplates(): جلب القوالب
- addTemplate(template): إضافة قالب
- deleteTemplate(id): حذف قالب
